# 00-PREREQUISITES.md

## Prerequisites

Before starting, make sure you have the following:

### Software Requirements

- **Node.js** (v18 or higher) OR **Bun** (v1.3 or higher)
- **Git** installed on your machine
- **Code Editor** (VS Code recommended)
- **Terminal/Command Line** access

### Accounts You Need to Create

Create free accounts on these platforms (you can do this as you go):

1. **[Neon](https://neon.com)** - PostgreSQL database hosting
2. **[Unsplash](https://unsplash.com)** - Stock photo API
3. **[UploadThing](https://uploadthing.com)** - File upload service
4. **[Replicate](https://replicate.com)** - AI image generation (requires $2+ billing)
5. **[Stripe](https://stripe.com)** - Payment processing
6. **[GitHub](https://github.com)** - Code hosting & OAuth
7. **[Google Cloud](https://console.cloud.google.com)** - Google OAuth

### Estimated Time

- **Total setup time:** ~90 minutes
- **Prerequisites setup:** ~15 minutes (creating accounts)

### What You'll Build

A full-stack Canva clone with:
- Image editing and AI generation
- User authentication (Email, GitHub, Google)
- Payment processing with Stripe
- Template system
- Cloud file storage
